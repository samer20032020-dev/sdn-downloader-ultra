# 🎬 برنامج مُنزّل الفيديو والصوت الشامل (OmniDownloader)

تطبيق سطح مكتب جديد بامتياز، ذو واجهة عربية جذابة وسهلة الاستخدام يتيح لك تحميل مقاطع الفيديو أو الصوتيات من أي موقع في العالم (إنستغرام، يوتيوب، فيسبوك، تيك توك، تويتر، بينتريست، وغيرها من مئات المنصات والمواقع).

---

## 🌟 مميزات البرنامج
1. **دعم كوني (أي رابط من أي موقع)**: بفضل محرك `yt-dlp` المطور يتم جلب الفيديوهات من كل المنصات الشهيرة والغير معروفة.
2. **استخراج الخيارات فوراً**: بمجرد وضع الرابط يظهر لك صورة المعايرة والعنوان والمدة مع زرين مخصصين:
   - 🎥 **فيديو بصوت**: اختر الجودة المناسبة (4K, 1080p, 720p, 480p, إلخ).
   - 🎵 **صوت فقط**: للتحميل بصيغة MP3 أو M4A بأعلى جودة.
3. **تحديد مسار الحفظ**: يمكنك اختيار المجلد الذي تريد حفظ الملفات فيه بسهولة.
4. **شريط تقدم ومتابعة حية**: يوضح سرعة التحميل بالحجم والميغابايت والوقت المتبقي.

---

## 🚀 طريقة التشغيل
يمكنك تشغيل البرنامج بأحد الطرق التالية:

### الطريقة المباشرة (الأسهل):
قم بالضغط مرتين (Double Click) على الملف:
👉 `تشغيل_البرنامج.bat`

### أو عبر سطر الأوامر:
```bash
python main.py
```

## 📦 المكتبات المعتمدة
- **Python 3**
- **CustomTkinter** (للواجهة المظلمة الحديثة)
- **yt-dlp** (لجلب وتنزيل الفيديوهات والصوتيات)
- **Pillow / Requests** (لمعاينة الصور المعروضة)
